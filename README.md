# Space-Execution
Space Defender - 2D аркадный шутер на Pygame. Управляйте космическим кораблём, уничтожайте волны вражеских НЛО и астероидов, собирайте аптечки. Уклоняйтесь от столкновений, используйте ограниченный боезапас с перезарядкой. Цель: набрать очки и не пропустить более 5 врагов. Полная перезагрузка по R.
